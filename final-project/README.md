# Project description

## Какие технологии применялись в работе над проектом

- Create React App
- Function components + React Hooks
- React-router 5
- Метод setTimeout для имитации асинхронного запроса
- LocalStorage
- Flex 
- Адаптивная верстка
- Анимация (@keyframes, transition)

## Проект состоит из следующих страниц
- Главная страница (общий доступ). Кнопка «Выбрать тур» ведет на страницу «Туры».
- Туры (общий доступ). Кнопка «Заказать» ведет на форму для отправки заявки на тур.
- Авторизация для доступа в CRM в качестве администратора.
- Форма для отправки заявки на тур (общий доступ). С помощью LocalStorage данные по заявке сохраняются, и администратор может их просмотреть в CRM после авторизации.
- CRM (приватный доступ). Здесь можно посмотреть поступившие заявки, а также удалить одну из них или все.
